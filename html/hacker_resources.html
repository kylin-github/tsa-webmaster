A Guide to Virtual Hackathons

Is this your first ever hackathon? Or maybe it's your first ever virtual hackathon? If so, this guide will help you make the most of it!
Codecademy

Don't know how to code? No problem! Codecademy offers free coding lessons that teach you to code in countless languages.
Udacity

Udacity is another great resource to learn the necessary skills to ace your hackathon project! It offers a wide variety of course that are 100% online.
Python

When chooosing which programming language to use, Python is a great option. Its intuitive design makes it perfect for beginners, while the extensive functionality it offers through its many libraries makes it a favorite for pros.

Github

Coding in teams can be difficult sometimes, especially in a virtual environment. Github is a tool that makes the process so much easier by providing a way for many people to work on a project together.
